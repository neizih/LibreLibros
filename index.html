Motivo

Muchas de las páginas actuales tienen un límite de descarga, requieren de registro e inicio de sesión, tienen publicidad, entre otras cosas. Se busca ofrecer la mejor experiencia posible para descargar y leer libros en una página de muy poco peso!

¡Aporta¡

Hay varias formas de ayudar al proyecto, ya sea compartiendo un libro o usando el enlace de donación, toda donación irá para la transcripción de libros para todos sus formatos. Todo usuario es libre de aportar un libro a través de Github. Además, para motivar el proyecto, todo usuario que aporte con un libro, podrá dejar un enlace propio de donación. Clickaquípara ver cómo aportar un libro.
